Introdução e Estrutura HTML

Quais elementos básicos do HTML?

  • HTML é construído a partir de elementos, que são estruturas definidas por tags. As tags são cercadas por colchetes angulares (< >). Alguns elementos básicos são:

    • HTML

      • Define o início e o fim do documento HTML.
      • A tag raiz, engloba todo o HTML. Ela envolve as tags 'head' e 'body'.

    • HEAD

      • Contém informações sobre o documento, como título, metadados e links para arquivos externos.
      • Aqui é onde você coloca metadados e informações sobre o documento, como o título da página, links para folhas de estilo CSS e outros recursos.

    • BODY

      • Contém todo o conteúdo visível na página, como texto, imagens e links.
      • A taga que engloba todo conteúdo visível da sua página, como texto, imagens e outros elementos. É a parte da página que os usuários interagirão.

Atributos e Tags

  • As tags HTML podem ter atributos, que fornecem informações adicionais sobre o elemento. Por exemplo, a tag 'a' (link) pode ter o atributo 'href' para indicar a URL do link. A tag 'img' (imagem) usa o atributo 'src' para especificar o caminho da imagem.

Texto e Formatação

Parágrafos e Quebras de Linha

  • A tag 'p' é usada para criar parágrafos de texto. Ela automaticamente insere espaço em branco antes e depois do conteúdo, tornando o texto mais legível. Já a tag 'br' é usada para criar uma quebra de linha dentro de um parágrafo, ideal para separar linhas em endereços ou poesias.

Listas e Alinhamento

  • Listas não ordenadas:

    • As listas não ordenadas são ideais para representar itens que não têm uma sequência específica. Elas são criadas usando a tag 'ul' , e cada item da lista é marcado com a tag 'li'. O navegador renderizará cada item com um marcador padrão, como um ponto ou um círculo.

  • Listas Ordendas:

    • As listas ordenadas são usadas quando você deseja mostrar uma sequência de itens. Elas são semelhantes às listas não ordenadas, mas em vez de marcadores, os itens são numerados automaticamente. Use a tag 'ol' para criar uma lista ordenada e a tag 'li' para cada item.

  • Listas Aninhadas:

    • As listas aninhadas permitem que você crie estruturas hierárquicas de informações. Você pode aninhar uma lista dentro de outra, usando a tag 'ul' ou 'ol' para a lista externa e a tag 'li' para os itens externos e internos. Isso é útil para mostrar categorias e subcategorias, por exemplo.

  • Listas de Definição:

    • Além das listas não ordenadas e ordenadas, existe a lista de definição. Ela é criada com a tag 'dl' , que contém pares de termos ('dt') e suas definições ('dd') . Essas listas são frequentemente usadas para criar glossários ou explicações de conceitos.

Listas e Semântica:

  • Ao usar listas corretamente, você também está adicionando significado semântico ao seu conteúdo. Isso melhora a acessibilidade e a compreensão do conteúdo pelos mecanismos de busca.

Links e Âncoras

Criando links com a Tag 'a'

  • A tag 'a' (âncora) é usada para criar links em HTML. Você pode usar essa tag para vincular páginas internas e externas. O atributo href é utilizado para definir o destino do link. Quando um usuário clica em um link, ele é direcionado para a URL especificada.

Vinculando Páginas Internas e Externas

  • Para vincular a uma página interna, como uma página em seu próprio site, você pode usar um caminho relativo, como pagina.html. Para vincular a uma página externa, como um site externo, você pode usar a URL completa, como https://www.exemplo.com.

Links de E-mail e Links para Download

Imagens e Multimídia

Inserindo Imagens com a Tag "img"

  • A tag 'img' é usada para exibir imagens em suas páginas. Você definirá o caminho da imagem no atributo src e poderá especificar um texto alternativo usando o atributo alt. Isso é fundamental para acessibilidade e também é exibido quando a imagem não pode ser carregada.

Especificando o Tamanho da Imagem

  • É possível controlar o tamanho da imagem com os atributos width (largura) e height (altura) . É importante manter as proporções da imagem para que ela não fique distorcida.

Controles de Mídia e Opções Avançadas

  • A tag 'video' e 'audio' permite que você adicione controles para que os usuários possam reproduzir, pausar e ajustar o volume dos vídeos e áudios. Além disso, você pode definir opções avançadas, como loop (repetição), autoplay (reprodução automática) e mais.

Mídia Incorporada a Terceiros

  • Você pode incorporar conteúdo de plataformas externas, como vídeos do YouTube, utilizando códigos de incorporação fornecidos por essas plataformas. Isso permite que você insira facilmente conteúdo de alta qualidade em suas páginas

Tabelas

Construindo Tabelas com a Tag 'table'

  • Você definirá as linhas da tabela com a tag 'tr' (table row) e as células com as tags 'th' (table header) para cabeçalhos e 'td' (table data) para dados.

    • Exemplo:

      • Nome Idade Cidade
      Maria 18 Anos Salvador, Bahia
      Lucas 20 Anos Vitória da Conquista

Mesclando Células e Ajustando o Layout

  • Você pode mesclar células de tabelas usando os atributos rowspan (mesclagem de linhas) e colspan (mesclagem de colunas). Isso é útil para criar células que abrangem várias linhas ou colunas. Além disso, você pode ajustar o layout da tabela usando atributos como border, cellpadding e cellspacing.

    • Exemplo:

      • Nome Idade Cidade
      Maria 18 Anos Salvador, Bahia
      Lucas 20 Anos Vitória da Conquista

Cabeçalhos de Tabela e Escopos

  • A tag 'th' é usada para criar cabeçalhos de tabela. Você pode usar o atributo scope para indicar se o cabeçalho se aplica a uma coluna (col) ou a uma linha (row), melhorando a acessibilidade.

    • Exemplo:

      • Segunda-Feira Terça-Feira Quarta-Feira
      Manhã Café Chá Suco
      Tarde Energético Refri Cerveja

Formulários e Entradas

Criando Formulários com a Tag 'form'

  • A tag 'form' é usada para criar formulários em HTML. Ela envolve elementos de entrada e define a ação que será executada ao enviar o formulário. O atributo action especifica o URL para o processamento do formulário e os métodos GET e POST determinam como os dados do formulário serão enviados.

Tipos de Entrada

  • Existem vários tipos de elementos de entrada que você pode adicionar aos formulários, como 'input', 'textarea', 'select', entre outros . O elemento 'input' é especialmente versátil, permitindo que você crie campos de texto, botões de rádio, caixas de seleção e mais.

Enviando Dados para o Servidor

  • Ao enviar um formulário, os dados inseridos pelos usuários podem ser enviados para um servidor web. O método GET anexa os dados ao URL da ação, enquanto o método POST envia os dados de forma mais segura e independente do URL.

Validação de Entrada

  • Você pode adicionar validações aos campos de entrada para garantir que os dados fornecidos pelos usuários estejam no formato correto. Isso ajuda a evitar erros e melhorar a qualidade dos dados enviados

Elementos Semânticos

Qual a importância dos Elementos Semânticos?

  • Os elementos semânticos em HTML são tags que têm significados específicos para os navegadores e mecanismos de busca. Eles descrevem o conteúdo de uma página de forma mais precisa, tornando-a mais compreensível tanto para humanos quanto para máquinas.

Tags como 'header', 'nav', 'article' e 'footer'

  • A tag 'header' representa o cabeçalho da página ou de uma seção específica, contendo elementos como logotipos, títulos e informações de navegação.
  • A tag 'nav' é usada para definir uma seção de navegação, normalmente contendo links para outras páginas ou partes do site.
  • A tag 'article' é utilizada para envolver um conteúdo independente e autocontido, como postagens de blog, notícias ou histórias.
  • A tag 'footer' representa o rodapé da página ou de uma seção, contendo informações de direitos autorais, links úteis e outros detalhes.

Melhorando a Acessibilidade e a Estrutura do Conteúdo

  • O uso adequado de elementos semânticos não apenas melhora a compreensão do conteúdo pelos motores de busca, mas também contribui para uma melhor acessibilidade. Leitores de tela e outros dispositivos auxiliares podem utilizar essas tags para compreender a estrutura e o significado do conteúdo, tornando-o mais acessível a pessoas com deficiências visuais.
